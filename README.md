# 📊 India Retail Demand Risk Monitor

> AI-powered decision support dashboard for Indian retail businesses using macroeconomic indicators, Python, Power BI and AI-assisted business insights.

---

## 📖 Project Overview

This project helps retail businesses in India monitor macroeconomic conditions that influence consumer demand.

Using official economic indicators from the FRED API, the system calculates a Retail Demand Risk Score and presents actionable insights through an interactive Power BI dashboard.

An AI-assisted analysis layer is then used to explain trends and recommend business actions in natural language.

---

## 🎯 Business Problem

Retail businesses are heavily affected by macroeconomic changes such as:

- Inflation
- Interest Rates
- Exchange Rates
- GDP Growth
- Money Supply

Most businesses react after demand changes.

This project aims to provide an early warning system to help decision-makers anticipate demand shifts before they occur.
